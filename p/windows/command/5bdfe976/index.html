netsh(Network Shell) 是一个windows系统本身提供的功能强大的网络配置命令行工具

修改mtu

C:\WINDOWS\system32>netsh
netsh>interface
在将来的 Windows 版本中，Microsoft 可能会删除 TCP/IP 的 Netsh 功能。

如果你当前使用 Netsh 配置和管理 TCP/IP，Microsoft 建议你转移到 Windows PowerShell。

要查看用于管理 TCP/IP 的命令列表，请在 Windows PowerShell 提示符处键入 Get-Command -Module NetTCPIP。

有关 TCP/IP 的 PowerShell 命令的其他信息，请访问 https://go.microsoft.com/fwlink/?LinkId=217627。
netsh interface>ipv4
netsh interface ipv4>show subinterface

   MTU  MediaSenseState   传入字节  传出字节      接口
------  ---------------  ---------  ---------  -------------
4294967295                1          0    3412447  Loopback Pseudo-Interface 1
  1500                1  3162790465  581193074  以太网 2

netsh interface ipv4>set subinterface "以太网 2" mtu=1400
确定。

netsh interface ipv4>

也可以直接输入以下命令

netsh interface ipv4 set subinterface "以太网 2" mtu=1400

初始化网络状态命令

netsh winsock reset
