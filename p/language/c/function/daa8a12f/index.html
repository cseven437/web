C 库函数char *strcpy(char *dest, const char *src)把 src 所指向的字符串复制到 dest。

需要注意的是如果目标数组 dest 不够大，而源字符串的长度又太长，可能会造成缓冲溢出的情况。

C 库函数char *strncpy(char *dest, const char *src, size_t n)把 src 所指向的字符串复制到 dest，最多复制 n 个字符。当 src 的长度小于 n 时，dest 的剩余部分将用空字节填充。

#include <stdlib.h>

char *strcpy(char *dest, const char *src)
char *strncpy(char *dest, const char *src, size_t n)

参数

  • dest – 指向用于存储复制内容的目标数组。
  • src – 要复制的字符串。
  • n – 要从源中复制的字符数。

返回值

该函数返回一个指向最终的目标字符串 dest 的指针。

#include <stdio.h>
#include <string.h>
 
int main()
{
   char src[40];
   char dest[100];
  
   memset(dest, '\0', sizeof(dest));
   strcpy(src, "This is runoob.com");
   strcpy(dest, src);
 
   printf("最终的目标字符串： %s\n", dest);
   
   return(0);
}

让我们编译并运行上面的程序，这将产生以下结果：

最终的目标字符串： This is runoob.com
