在将来的 Windows 版本中，Microsoft 可能会删除 TCP/IP 的 Netsh 功能。

如果你当前使用 Netsh 配置和管理 TCP/IP，Microsoft 建议你转移到 Windows PowerShell。

要查看用于管理 TCP/IP 的命令列表，请在 Windows PowerShell 提示符处键入 Get-Command -Module NetTCPIP。

有关 TCP/IP 的 PowerShell 命令的其他信息，请访问 https://go.microsoft.com/fwlink/?LinkId=217627。
