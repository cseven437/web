问题：linux中vmware出现：Network configuration is missing. Ensure that /etc/vmware/networking exists.的处理方式 环境：archlinux 时间：2020-04-12

安装aur源的vmware 15后，一直无法联网

终端输入：vmware-netcfg ，来调用vmware-netcfg来初始化vmware网络模块

这时还会提示：Network configuration is missing. Ensure that /etc/vmware/networking exists.

解决方法：touch /etc/vmware/x ; vmware-networks --migrate-network-settings /etc/vmware/x ; rm /etc/vmware/x

这条命令是人工重建vmware-networks模块

再次输入：vmware-netcfg，看看成功调用了vmware网络模块了
