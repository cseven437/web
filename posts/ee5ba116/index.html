使用 sublime记录

自定义配置文件

Preferences-->Settings-->

{
    // 字号
    "font_size": 11,
    // 使用空格替换tab键
    "translate_tabs_to_spaces": true,
    // 状态栏显示文件结尾格式 unix windows
    "show_line_endings": true,
    // 状态栏显示编码格式 utf-8
    "show_encoding": true,
    // 中文字体显示异常时加这个配置
    "font_options": ["gdi"] 
}

查看当前使用的字体

使用快捷键 C-` 打开console，输入下面的命令

view.settings().get('font_face')
